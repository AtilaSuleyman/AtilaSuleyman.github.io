Latest Projects

Own Project - Events Locator

Finds the closest events given a coordinate and returns events and their cheapest ticket

Java

Built in Java, This program builds a world which is essentially a grid of 20 by 20. The world is initialised with random events which have ID's and also holds a set of randomly generated tickets with randomly generated prices.

The user can type in a coordinate the wish to view events for, the program will then use Manhattan Distance to find the 5 closest events to the location along with their distance and their cheapest ticket and give the information back to the user.
